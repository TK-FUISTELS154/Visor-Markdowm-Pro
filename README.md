# Visor-Markdowm-Pro

**Repositorio:** [TK-FUISTELS154/Visor-Markdowm-Pro](https://github.com/TK-FUISTELS154/Visor-Markdowm-Pro)

**Autor:** TK-FUISTELS154

**Descripción breve:** Una pequeña aplicación web hecha en HTML/CSS/JavaScript para visualizar archivos Markdown localmente o en línea, con una interfaz sencilla.

---

## Descripción del proyecto

Este proyecto “Visor-Markdowm-Pro” proporciona una interfaz ligera para cargar y visualizar archivos `.md` (Markdown). Está construido usando tecnologías web estándar (HTML, CSS, JS) para permitir:

* Cargar un archivo Markdown desde el equipo local o mediante enlace.
* Renderizar el contenido Markdown como HTML legible en el navegador.
* Un entorno simple que sirve como base para añadir funcionalidades (temas, vista previa, arrastrar y soltar, etc.).
* Ideal para documentación, notas, demostraciones rápidas sin necesidad de instalar herramientas complejas.

El lenguaje predominante: JavaScript (~47.9 %), CSS (~33.1 %) y HTML (~19.0 %) según el análisis del repositorio.

## Estructura del repositorio

A continuación se presenta la organización de ficheros/directorios del proyecto en su forma actual:

```
/Visor-Markdowm-Pro
├── css/
│   └── [ficheros CSS para estilos]
├── js/
│   └── [ficheros JavaScript para funcionalidad]
└── index.html
```

* `css/` – carpeta que contiene los estilos visuales del proyecto.
* `js/` – carpeta que contiene los scripts de funcionalidad (por ejemplo, carga de archivo, renderizado de Markdown).
* `index.html` – página principal de la aplicación web, que carga los estilos y scripts y muestra la interfaz al usuario.

---

## Uso

1. Clonar o descargar el repositorio.

   ```bash
   git clone https://github.com/TK-FUISTELS154/Visor-Markdowm-Pro.git
   ```
2. Abrir el archivo `index.html` en un navegador web (localmente).
3. En la interfaz, selecciona un archivo `.md` o introduce un enlace, y visualízalo convertido a formato HTML.
4. (Opcional) Personaliza estilos o añade funcionalidades según tus necesidades.
